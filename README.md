# 🌿 Plant Doctor AI

Plant Doctor AI is a deep learning-based project that detects tomato plant diseases using image classification.

## 🔍 Features
- Detects multiple tomato leaf diseases
- Uses MobileNetV2 (Transfer Learning)
- Provides treatment & prevention suggestions
- Achieves ~96% accuracy

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- Google Colab
- PlantVillage Dataset

## 📊 Model Performance
- Training Accuracy: ~96%
- Validation Accuracy: ~96%

## 📁 Files
- plant_doctor.ipynb → Main code
- plant_doctor_model.h5 → Trained model
- plant_doctor_result.png → Sample output

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload kaggle.json
3. Run all cells step by step
4. Upload leaf image for prediction

## 📌 Note
This model is trained only for tomato plant diseases.
